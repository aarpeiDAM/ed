Hemos instalado el jdk de java8, para ello hemos añadido el repositorio de java con el comando: sudo apt-add-repository ppa:webupd8team/java
A continuación actualizamos los repositorios: sudo apt upadte
Por último los instalamos: sudo apt install oracle-java8-installer 

Cuando compilamos en java un fichero tendremos el fichero de partida y obtendremos uno nuevo con el nombre "<nombre>.class", en dicho archivo estara el código ejecutable en jre.
